# Research Notes

## Topic
The Secret Behind Recommendation Systems: How Netflix, Spotify, and TikTok Recommend What You Like

## Why I chose this topic
I chose this topic because recommendation systems are something we use almost every day.

- Netflix recommends movies and shows
- Spotify recommends songs and playlists
- TikTok recommends videos on the For You page

It feels like these apps know what we like, but really they are using data, algorithms, and machine learning.

## What is a recommendation system?
A recommendation system is software that suggests content to users.

It tries to predict what someone may enjoy based on their actions.

For example:

- If someone watches many cooking videos, TikTok may show more cooking videos.
- If someone skips a song quickly, Spotify may understand that the user does not like that type of song.
- If someone watches many mystery shows, Netflix may recommend more mystery or crime shows.

## Why this is related to software development
Recommendation systems are related to software development because developers build the systems that collect data, analyze patterns, and show personalized results.

These systems use:

- Algorithms
- Data collection
- Machine learning
- User behavior tracking
- User experience design

This shows that software development is not only about writing code. It is also about understanding users and designing apps that feel personal.

## Netflix
Netflix recommends movies and shows based on what users watch, search for, rate, and what similar users enjoy.

For example:

- If a user watches a lot of romance movies, Netflix may recommend more romance movies.
- If a user watches crime shows, Netflix may suggest similar mystery or crime shows.
- Netflix also decides which titles appear higher on the screen.

Source:
https://help.netflix.com/en/node/100639

## Spotify
Spotify recommends music based on listening habits.

It can learn from:

- Songs users play
- Songs users skip
- Songs users save
- Searches
- Playlists

For example:

- If someone listens to sad songs at night, Spotify may suggest similar songs.
- If someone saves many songs by the same artist, Spotify may recommend more music from that artist.

Source:
https://www.spotify.com/safetyandprivacy/understanding-recommendations

## TikTok
TikTok uses the For You feed to recommend videos.

It learns from:

- Likes
- Comments
- Follows
- Videos watched
- Videos watched until the end
- Videos marked as not interested

For example:

- If a user watches makeup videos fully, TikTok may show more makeup videos.
- If a user skips sports videos quickly, TikTok may show fewer sports videos.

Source:
https://support.tiktok.com/en/using-tiktok/exploring-videos/how-tiktok-recommends-content

## Benefits
Recommendation systems can be helpful because they:

- Save time
- Help users discover new content
- Make apps feel more personal
- Improve user experience
- Help users find movies, songs, and videos they may enjoy

## Problems and concerns
Recommendation systems also have problems:

- Apps may collect too much personal data
- Users may spend too much time scrolling
- People may only see content similar to what they already like
- This can create filter bubbles
- Users may not know why something is being recommended

## My opinion
I think recommendation systems are useful because they make apps easier and more fun to use.

However, they should be designed responsibly.

Users should understand how their data is being used, and companies should not only use recommendations to keep people addicted to their apps.

Overall, recommendation systems are powerful software tools because they shape what we watch, listen to, and enjoy every day.
